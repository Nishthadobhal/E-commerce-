# 🛒 Cartify - E-commerce Website

A full-stack e-commerce web application built using PHP and MySQL.
It allows users to browse products, manage cart, and place orders, with a dedicated admin panel for management.

---

## 🚀 Live Demo

👉 https://cartify-shop.42web.io/ecommerce

---

## ✨ Features

### 👤 User Side

* User Registration & Login
* Browse Products & Categories
* Search Functionality
* Add to Cart
* View Orders

### 🛠 Admin Panel

* Manage Products
* Manage Categories & Brands
* View Orders
* User Management

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, Bootstrap
* **Backend:** PHP
* **Database:** MySQL
* **Server:** Apache (XAMPP / InfinityFree)

## ⚙️ Setup Instructions

1. Clone the repository
2. Import the SQL file (`mystore.sql`) into phpMyAdmin
3. Configure database in `connect.php`
4. Run on localhost (XAMPP)
